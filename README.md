# NakuNodes

这是为ComfyUI设计的工具集，提供了一系列实用的图像处理和操作节点，让我们的操作变得更加直观方便

  特别鸣谢

  本项目中的 FastCanvas 节点基于 GitHub 作者 @LAOGOU-666 (https://github.com/LAOGOU-666) 开源的 fastcanvas 节点进行开发，在此表示感谢！

  安装说明

   1. 确保已安装ComfyUI
   2. 将此仓库克隆到ComfyUI的custom_nodes目录下：

   1 cd ComfyUI/custom_nodes
   2 git clone https://github.com/naku-yh/ComfyUI-NakuNode

   3. 安装依赖：

   1 pip install -r requirements.txt

  使用方法

   1. 启动ComfyUI
   2. 在右键添加节点中，您可以在"NakuNodes"类别下找到所有工具节点
   3. 将需要的节点拖入工作区并连接使用

  节点说明


  图像尺寸调整节点
  !Image (./assets/size.jpg)
   - 调整图像尺寸，支持多种预设比例（1:1, 3:2, 3:4, 16:9等）
   - 可以保持原始比例或自由调整


  FastCanvas画布节点
  !Image (./assets/FastCanvas.png)
  > * 1.支持实时调整构图输出图像和选中图层遮罩
  > * 2.支持批量构图，切换输入图层继承上个图层的位置和缩放
  > * 3.支持限制画布窗口视图大小的功能，不用担心图片较大占地方了
  > * 4.图层支持右键辅助功能，变换，设置背景等，点开有惊喜
  > * 5.支持通过输入端口输入图像，支持无输入端口独立使用


  NAKU图像标注节点V2.0
   - 基于输入图像进行标注和绘制
   - 支持画笔、橡皮擦和方框工具
   - 撤销和重做功能
   - 多种颜色选择
   - 可调节画笔大小
   - 支持加载上游图像进行标注

  NAKU简易画板
   - 独立绘图功能，无需输入图像
   - 预设画板尺寸：1:1, 3:2, 3:4, 16:9
   - 支持横屏/竖屏模式（自动交换宽高）
   - 可选择背景颜色：白色、黑色、灰色
   - 画笔、橡皮擦和方框工具
   - 撤销和重做功能
   - 可选mask输入接口

  文本拆分节点
   - 用于拆分多行文本到可选选项
   - 支持格式：序号【模型名称】/模型提示词
   - 可选择是否仅输出提示词（提取"/"后内容）
   - 提供选中文本、选项列表和选中索引输出

  文本选择节点
   - 允许从选项列表中进行选择
   - 提供可视化界面进行文本选项选择
   - 可选择是否仅输出提示词（提取"/"后内容）

  高级文本拆分节点
   - 拆分多行文本成可选选项
   - 提供第一个选项、选项列表和索引输出
   - 支持刷新选项功能

  文本选择器节点
   - 允许从列表中选择文本选项
   - 支持通过索引选择特定选项
   - 提供选中文本和索引输出

  动态文本拆分与选择节点
   - 用于拆分和选择文本，适用于Lora提示词筛选器
   - 支持按索引选择选项
   - 可选择是否仅输出提示词（提取"/"后内容）
   - 提供选中文本、全部选项和实际索引输出

  保存图像节点
   - 用于保存处理后的图像
   - 支持多种输出格式（PNG/JPG/WebP）
   - 自定义文件名前缀，支持时间戳、日期等变量
   - 可设置输出质量（仅JPG/WebP格式）
   - 可设置自定义保存路径
